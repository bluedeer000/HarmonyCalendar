# 真机回归计划

## 目标

在提交 AppGallery 审核前，用真实 HarmonyOS 手机完成一轮最小可交付回归。

## 测试环境

- HarmonyOS 手机 1 台以上
- 可用网络环境
- 至少 1 个可登录的 CalDAV 测试账号

## 核心测试项

### 启动与基础体验

- 应用首次启动正常
- 首页可正常渲染
- 日/周/月/列表视图切换正常
- 搜索输入和清空正常

### 账号与连接

- 新增 CalDAV 账号
- 测试连接成功
- 错误用户名或密码时提示合理
- 根地址与 collection URL 两种方式都验证

### 数据同步

- 首次同步成功
- 新建日程后同步到远端
- 编辑日程后同步到远端
- 删除日程后同步删除远端
- 远端修改后本地再次同步能看到更新

### 提醒

- 允许提醒权限
- 创建提醒
- 修改提醒
- 删除提醒
- 到点提醒可实际触发

### 兼容边界

- 带时区事件
- 带重复规则事件
- 多提醒事件
- 带参会人与链接的事件

## 高风险项

- `PROPFIND` / `REPORT` / `PUT` / `DELETE` 在真机网络栈上的实际表现
- 第三方 CalDAV 服务的认证与重定向
- 提醒权限和提醒触发的系统限制
- 重复规则边界处理

## 回归完成标准

- 核心测试项全部通过
- 无阻塞级崩溃
- 无“无法同步”类高频故障
- 审核员可在 5 分钟内完成一轮关键路径验证
