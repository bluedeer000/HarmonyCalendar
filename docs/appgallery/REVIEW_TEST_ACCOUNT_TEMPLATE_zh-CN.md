# 审核测试账号说明模板

用途：提交 AppGallery Connect 时，将本文件中的占位符替换后，直接复制到“测试信息”或“审核备注”相关区域。

## 可直接提交版

请使用以下审核测试账号验证应用核心功能：

- 测试服务地址：`[填写你的 CalDAV 服务地址]`
- 测试用户名：`[填写测试用户名或邮箱]`
- 测试密码：`[填写应用专用密码]`

审核步骤：

1. 启动应用。
2. 进入账号配置页。
3. 填写上述服务地址、用户名和应用专用密码。
4. 点击“测试连接”，确认连接成功。
5. 返回主界面后执行同步。
6. 查看日视图、周视图、月视图、列表视图中的日程显示是否正常。
7. 新建一个带提醒的日程并执行同步。
8. 编辑该日程并再次同步。
9. 删除该日程并确认同步删除。

补充说明：

- 本应用不提供自有云服务，仅连接用户主动配置的第三方 CalDAV 服务。
- 请优先验证连接、同步、新建、编辑、删除和提醒相关功能。
