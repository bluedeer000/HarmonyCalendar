# AppGallery 中文素材草案

## 应用名称

Macto 日历

## 一句话简介

支持 CalDAV 同步的 HarmonyOS 日历应用。

## 应用简介

Macto 日历是一款面向 HarmonyOS 手机的日历应用，支持日视图、周视图、月视图和列表视图，帮助你统一管理日程、提醒和重复事件。应用支持连接兼容 CalDAV 的日历服务，并将远端日程同步到本地。

## 详细介绍

Macto 日历提供日视图、周视图、月视图和列表视图，支持搜索、提醒、重复规则、分类颜色、冲突提示和本地同步状态展示。你可以连接兼容 CalDAV 的日历服务，查看、创建、编辑和删除日程，并通过本地提醒及时收到待办通知。

当前版本重点能力：

- 支持 CalDAV 账号配置与连接测试
- 支持日程的新增、编辑、删除与同步
- 支持日、周、月、列表多视图切换
- 支持标题、地点、备注搜索
- 支持提醒、第二提醒和基础重复规则
- 支持同步状态展示和本地冲突提示

## 审核备注建议

- 应用不提供自有云服务。
- 应用仅连接用户主动配置的第三方 CalDAV 服务。
- 如审核需要测试账号，请提供可用的 CalDAV 测试服务地址和测试凭据。
